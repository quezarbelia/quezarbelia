## `> project --featured`
 
### 📊 Plataforma Fintech — Arquitectura Orientada a Datos
 
> _Sistema de gestión financiera personal con motor analítico integrado y modelo de negocio validado por métricas._
 
**Problema que resuelve:** La mayoría de herramientas financieras capturan datos pero no generan inteligencia accionable. Este sistema cierra esa brecha convirtiendo transacciones en decisiones.
 
#### Arquitectura & Stack
 
```
┌─────────────────────────────────────────────────────────┐
│                    CAPA DE PRESENTACIÓN                  │
│            Dashboard Tableau  ·  Reportes PDF            │
├─────────────────────────────────────────────────────────┤
│                    CAPA DE NEGOCIO                       │
│     Motor de Análisis Predictivo  ·  Reglas Financieras  │
├─────────────────────────────────────────────────────────┤
│                    CAPA DE DATOS                         │
│         ETL Pipeline  ·  Modelado Relacional SQL         │
├─────────────────────────────────────────────────────────┤
│                    CAPA DE SISTEMA                       │
│              Java Backend  ·  Ubuntu Server              │
└─────────────────────────────────────────────────────────┘
```
 
#### Decisiones técnicas clave
 
- **Modelado dimensional (SQL):** Esquema estrella para análisis de patrones de gasto histórico con queries optimizadas sub-100ms en conjuntos de datos de +500K registros.
- **ETL automatizado:** Pipeline de normalización e integración de fuentes financieras heterogéneas con validación de integridad referencial.
- **Dashboard analítico (Tableau):** KPIs en tiempo real — flujo de caja, proyecciones de ahorro, alertas de anomalías por categoría.
- **Flujo de usuario basado en métricas:** Cada decisión de UX respaldada por datos de conversión y retención.
 
#### Modelo de negocio
 
| Métrica | Indicador |
|---|---|
| Segmento objetivo | Profesionales 25-40 con gestión activa de finanzas |
| Propuesta de valor | Inteligencia financiera, no solo registro |
| Monetización | Freemium con análisis predictivo premium |
| Diferenciador técnico | Motor SQL propio vs. dependencia de APIs costosas |
